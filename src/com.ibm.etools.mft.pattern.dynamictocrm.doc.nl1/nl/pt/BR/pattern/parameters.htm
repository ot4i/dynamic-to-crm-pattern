Parâmetros para o Padrão Saída de Entidade de Conta do Microsoft Dynamics CRM: Entrada de Transporte Dinâmico

Seleção da Operação
Parâmetro de padrão Default Descrição
Operação

Escolha uma ou mais operações entre Criar, Recuperar, Atualizar e Excluir. É possível ter fluxos de mensagem relativos depois de gerar o padrão. Você deve selecionar pelo menos uma operação.
Transporte de Entrada Padrão
Parâmetro de padrão Default Descrição
Transporte de Entrada

Escolha um dos três transportes, MQ, File e HTTP, e configure os detalhes relativos nos Detalhes de Transporte abaixo.
Detalhes de Transporte

Detalhes de transporte de entrada baseados no transporte e na operação que você selecionou. Você deve fornecer o nome de fila de entrada para o transporte de MQ, diretório de entrada para transporte de Arquivo e o sufixo de URL para o transporte de HTTP.
Configuração do Esquema de Entrada de Usuário
Parâmetro de padrão Default Descrição
Configuração do Esquema

Especifique a mensagem de entrada e de saída, e o elemento global para a operação selecionada.

O elemento global que você escolhe para entrada é usado como o objeto de entrada no arquivo de mapa relevante. Por exemplo, se você escolher a operação Criar e especificar o esquema de entrada e o elemento global, o elemento global será usado como o objeto de entrada de um arquivo "Create.map". O esquema CRMCreate, contendo CRMCreate como seu elemento global, é usado como o objeto de saída.

O elemento global que você escolhe para saída é usado como o objeto de saída no arquivo de retorno de mapa relevante. Por exemplo, se você especificou um esquema de saída e elemento global para a operação Criar, o elemento global será usado como o objeto de saída de um arquivo "CreateReturn.map". O esquema CRMCreate, contendo CRMCreate como seu elemento global, é usado como o objeto de entrada.
Conexão do CRM
Parâmetro de padrão Default Descrição
URL da organização CRM

Este parâmetro padrão define a URL de serviço de organização do sistema CRM. Ela deve ser uma URL completa para o serviço da organização. Por exemplo, acessando o CRM nas instalações, http://ipaddress:port/orgName/XRMServices/2011/Organization.svc; acessando o CRM online, https://orgName.crm.dynamics.com/XRMServices/2011/Organization.svc.
Nome do usuário CRM

Este parâmetro padrão descreve o nome de usuário do CRM usado no sistema CRM.
Senha do usuário CRM

Este parâmetro padrão define a senha do usuário CRM que você especificou.
Domínio do CRM

Este parâmetro padrão define o domínio CRM que você usou.
Criação de log
Parâmetro de padrão Default Descrição
Efetuando login Limpo Este parâmetro padrão determina se a instância padrão inclui o subfluxo para criação de log.

Se esse parâmetro padrão for limpo, o subfluxo da criação de log não será incluído.

Se esse parâmetro padrão for selecionado, você deverá configurar valores para Gerenciador de Filas de Log e Fila de log.
Gerenciador de filas de log Este parâmetro padrão define o gerenciador de filas usado para a criação de log. É possível deixar esse parâmetro em branco se o gerenciador de filas do broker for usado para criação de log.

Esse parâmetro padrão só será requerido se a opção Criação de log requerida for selecionada.
Fila de Logs Este parâmetro padrão define a fila usada para a criação de log.

Esse parâmetro padrão só será requerido se a opção Criação de log requerida for selecionada.
Tratamento de erro
Parâmetro de padrão Default Descrição
Erro ao efetuar logon em Selecionado Este parâmetro padrão determina se a instância padrão inclui um código para colocar mensagens de erro em uma fila.

Se esse parâmetro padrão for selecionado, os valores deverão ser configurados para Gerenciador de filas de erro e Fila de erros.
Gerenciador de filas de erros Este parâmetro padrão define o gerenciador de filas usado para a criação de log.

Este parâmetro padrão só será necessário se a opção Mensagem de erro requerida for selecionada.
Fila de erros Este parâmetro padrão define a fila usada para as mensagens de erro.

Este parâmetro padrão só será necessário se a opção Mensagem de erro requerida for selecionada.

Todos os parâmetros de padrão são listados com seus valores-padrão. Onde não houver padrão indicado, o valor padrão será uma sequência vazia.
