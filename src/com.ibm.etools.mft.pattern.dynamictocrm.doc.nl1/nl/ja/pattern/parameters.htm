「Microsoft Dynamics CRM Account Entity 出力: 動的トランスポート入力」パターンのパラメーター

操作の選択
パターン・パラメーター デフォルト 説明
操作

「作成」、「取得」、「更新」、および「削除」から 1 つ以上の操作を選択します。パターンを生成した後で、関連するメッセージ・フローを含めることができます。少なくとも 1 つの操作を選択する必要があります。
パターン入力トランスポート
パターン・パラメーター デフォルト 説明
入力トランスポート

「MQ」、「ファイル」、および「HTTP」の 3 つのトランスポートから 1 つを選択して、下記の「トランスポート詳細」で関連する詳細を構成します。
トランスポート詳細

選択したトランスポートおよび操作に基づいてトランスポートの詳細を入力します。MQ トランスポートの場合は入力キュー名、ファイル・トランスポートの場合は入力ディレクトリー、および HTTP トランスポートの場合は URL 接尾部を指定する必要があります。
ユーザー入力スキーマ構成
パターン・パラメーター デフォルト 説明
スキーマ構成

選択した操作に対して、入力および出力のメッセージ・スキーマとグローバル・エレメントを指定します。

入力に対して選択したグローバル・エレメントは、関連マップ・ファイルで入力オブジェクトとして使用されます。例えば、「作成」操作を選択して入力スキーマとグローバル・エレメントを指定する場合、グローバル・エレメントは、「Create.map」ファイルの入力オブジェクトとして使用されます。グローバル・エレメントとして CRMCreate が入っている CRMCreate スキーマは、出力オブジェクトとして使用されます。

出力に対して選択したグローバル・エレメントは、関連マップ・リターン・ファイルで出力オブジェクトとして使用されます。例えば、「作成」操作に対して出力スキーマとグローバル・エレメントを指定した場合、グローバル・エレメントは、「CreateReturn.map」ファイルの出力オブジェクトとして使用されます。グローバル・エレメントとして CRMCreate が入っている CRMCreate スキーマは、入力オブジェクトとして使用されます。
CRM 接続
パターン・パラメーター デフォルト 説明
CRM 組織 URL

このパターン・パラメーターは、CRM システムの組織サービス URL を定義します。 これは、組織サービスの完全 URL でなければなりません。 例えば、CRM オンプレミスにアクセスする場合は http://ipaddress:port/orgName/XRMServices/2011/Organization.svc、CRM オンラインにアクセスする場合は https://orgName.crm.dynamics.com/XRMServices/2011/Organization.svc のようにします。
CRM ユーザー名

このパターン・パラメーターは、CRM システムで使用された CRM ユーザー名を記述します。
CRM ユーザー・パスワード

このパターン・パラメーターは、指定された CRM ユーザーのパスワードを定義します。
CRM ドメイン

このパターン・パラメーターは、使用された CRM ドメインを定義します。
ロギング
パターン・パラメーター デフォルト 説明
ロギングをオン クリア済み このパターン・パラメーターは、ロギング用のサブフローをパターン・インスタンスに含めるかどうかを決定します。

このパターン・パラメーターが非選択の場合、ロギング・サブフローは含まれません。

このパターン・パラメーターを選択する場合は、「ログ・キュー・マネージャー」と「ログ・キュー」の値を設定しなければなりません。
ログ・キュー・マネージャー このパターン・パラメーターは、ロギングに使用するキュー・マネージャーを定義します。ロギングにブローカー・キュー・マネージャーが使用される場合、このパラメーターをブランクのままにすることができます。

このパターン・パラメーターは、「必要なロギング」が選択されている場合にのみ必要です。

ログ・キュー このパターン・パラメーターは、ロギングに使用するキューを定義します。

このパターン・パラメーターは、「必要なロギング」が選択されている場合にのみ必要です。

エラー処理
パターン・パラメーター デフォルト 説明
エラー・ロギングをオン 選択済み このパターン・パラメーターは、キューにエラー・メッセージを入れるためのコードをパターン・インスタンスに含めるかどうかを決定します。

このパターン・パラメーターを選択する場合は、「エラー・キュー・マネージャー」および「エラー・キュー」値を設定する必要があります。

エラー・キュー・マネージャー このパターン・パラメーターは、エラー・ロギングに使用するキュー・マネージャーを定義します。

このパターン・パラメーターは、「必要なエラー・メッセージ」が選択されている場合のみ必要です。
エラー・キュー このパターン・パラメーターは、エラー・メッセージに使用するキューを定義します。

このパターン・パラメーターは、「必要なエラー・メッセージ」が選択されている場合のみ必要です。

すべてのパターン・パラメーターとそのデフォルト値をリストします。 デフォルトが示されていない場合、デフォルト値は空ストリングです。
