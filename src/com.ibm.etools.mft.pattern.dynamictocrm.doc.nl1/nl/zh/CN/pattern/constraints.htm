有关使用 Microsoft Dynamics CRM Account Entity output: Dynamic transport input 模式的约束

使用此模式时会有若干约束。

只有满足以下所有条件，才可以使用 Microsoft Dynamics CRM Account Entity output: Dynamic transport input 模式：

  • 该模式只能在 Microsoft Windows 环境中的代理上部署和运行，这是因为 .NETCompute 节点依赖于 .NET Framework 来连接到 Microsoft Dynamics CRM 系统。
  • 消息流实例和 BAR 文件包含明文形式的 Microsoft Dynamics CRM 连接凭证。如果这不适合于您的安全性需求，那么您必须修改消息流，以便 .NETCompute 节点可以通过安全机制获取这些详细信息。有关更多信息，请参阅应用 Microsoft Dynamics CRM Account Entity output: Dynamic transport input 模式之前要完成的任务。
  • 消息流实例只能设置和获取 Microsoft 帐户实体上的标准属性。您可以为定制属性修改和扩展模式创建的流实例。有关信息，请参阅应用 Microsoft Dynamics CRM Account Entity output: Dynamic transport input 模式之前要完成的任务。
