Microsoft Dynamics CRM Account Entity output: Dynamic transport input 模式的参数

选择操作
模式参数 缺省值 描述
操作

从“创建”、“检索”、“更新”和“删除”中选择一项或多项操作。在生成模式后，可以存在相对消息流。您必须至少选择一个操作。

模式输入传输
模式参数 缺省值 描述
输入传输

选择三种传输（MQ、文件和 HTTP）的其中一种，并在下面的“传输详细信息”中配置相对详细信息。
传输详细信息

基于所选传输和操作的输入传输详细信息。您必须提供输入队列名称（对于 MQ 传输）、输入目录（对于文件传输）和 URL 后缀（对于 HTTP 传输）。
用户输入模式配置
模式参数 缺省值 描述
模式配置

指定所选操作的输入和输出消息模式以及全局元素。

您为输入选择的全局元素将用作相关映射文件中的输入对象。例如，如果您选择了“创建”操作并指定了输入模式和全局元素，那么该全局元素将用作“Create.map”文件的输入对象。CRMCreate 模式（包含 CRMCreate 作为其全局元素）用作输出对象。

您为输出选择的全局元素将用作相关映射返回文件中的输出对象。例如，如果为“创建”操作指定了输出模式和全局元素，那么该全局元素将用作“CreateReturn.map”文件的输出对象。CRMCreate 模式（包含 CRMCreate 作为其全局元素）用作输入对象。
CRM 连接
模式参数 缺省值 描述
CRM 组织 URL

此模式参数定义 CRM 系统的组织服务 URL。它必须是指向组织服务的完整 URL。例如，要访问 CRM On-premise，是 http://ipaddress:port/orgName/XRMServices/2011/Organization.svc；要访问 CRM Online，是 https://orgName.crm.dynamics.com/XRMServices/2011/Organization.svc。
CRM 用户名

此参数描述您在 CRM 系统中使用的 CRM 用户名。
CRM 用户密码

此模式参数定义您指定的 CRM 用户的密码。
服务域

此模式参数定义使用的 CRM 域。
记录
模式参数 缺省值 描述
日志记录开启 已清除 此模式参数确定模式实例是否包括用于记录的子流。

如果已清除此模式参数，那么将不包括记录子流。

如果已选中此模式参数，那么您必须为日志队列管理器和日志队列设置值。
日志队列管理器 此模式参数定义用于记录的队列管理器。如果将代理队列管理器用于记录，那么此参数可保留为空。

仅当选择了需要记录时才需要此模式参数。
日志队列 此模式参数定义用于记录的队列。

仅当选择了需要记录时才需要此模式参数。
错误处理
模式参数 缺省值 描述
错误日志记录开启 已选中 此模式参数确定模式实例是否包括用于将错误消息放在队列中的代码。

如果选择了此模式参数，那么必须设置错误队列管理器和错误队列的值。
错误队列管理器 此模式参数定义用于错误记录的队列管理器。

仅当选择了需要错误消息时才需要此模式参数。
错误队列 此模式参数定义用于错误消息的队列。

仅当选择了需要错误消息时才需要此模式参数。
