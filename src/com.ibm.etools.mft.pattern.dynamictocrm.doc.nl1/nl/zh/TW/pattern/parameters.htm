Microsoft Dynamics CRM Account Entity 輸出：動態傳輸輸入型樣的參數

選取作業
型樣參數 預設值 說明
作業

從「建立」、「擷取」、「更新」及「刪除」中選擇一個以上作業。您可以在產生型樣之後具有相對訊息流程。您必須選取至少一項作業。
型樣輸入傳輸
型樣參數 預設值 說明
輸入傳輸

選擇三個傳輸（MQ、「檔案」及 HTTP）中的一個，並在下面的「傳輸詳細資料」中配置相對詳細資料。
傳輸詳細資料

根據您選取的傳輸與作業，輸入傳輸詳細資料。您必須提供 MQ 傳輸的輸入佇列名稱、「檔案」傳輸的輸入目錄及 HTTP 傳輸的 URL 字尾。
使用者輸入綱目配置
型樣參數 預設值 說明
綱目配置

指定所選取作業的輸入及輸出訊息綱目與廣域元素。

您選擇以輸入的廣域元素用作相對對映檔中的輸入物件。例如，如果您選擇「建立」作業並指定輸入綱目與廣域元素，則廣域元素用作 "Create.map" 檔的輸入物件。包含 CRMCreate 作為廣域元素的 CRMCreate 綱目用作輸出物件。

您選擇以輸出的廣域元素用作相對對映回覆檔中的輸出物件。例如，如果您指定「建立」作業的輸出綱目與廣域元素，則廣域元素用作 "CreateReturn.map" 檔的輸出物件。包含 CRMCreate 作為廣域元素的 CRMCreate 綱目用作輸入物件。
CRM 連線
型樣參數 預設值 說明
CRM 組織 URL

此型樣參數定義了 CRM 系統的組織服務 URL。它必須是組織服務的完整 URL。例如，內部存取 CRM，http://ipaddress:port/orgName/XRMServices/2011/Organization.svc；線上存取 CRM，https://orgName.crm.dynamics.com/XRMServices/2011/Organization.svc。
CRM 使用者名稱

此型樣參數會說明您在 CRM 系統中使用的 CRM 使用者名稱。
CRM 使用者密碼

此型樣參數會定義您指定的 CRM 使用者的密碼。
服務網域

此型樣參數會定義您使用的 CRM 網域。
記載
型樣參數 預設值 說明
開啟記載 已清除 此型樣參數決定型樣實例是否包括用於記載的子流程。

如果清除此型樣參數，則不會包括記載子流程。

如果選取此型樣參數，則必須設定日誌佇列管理程式及日誌佇列的值。
日誌佇列管理程式 此型樣參數定義用於記載的佇列管理程式。如果使用分配管理系統佇列管理程式以進行記載，則可以將此參數留為空白。

只有在選取需要記載時，才需要此型樣參數。
記載佇列 此型樣參數定義用於記載的佇列。

只有在選取需要記載時，才需要此型樣參數。
錯誤處理
型樣參數 預設值 說明
開啟記載時發生錯誤 已選取 此型樣參數決定型樣實例是否包括用來將錯誤訊息放置於佇列的程式碼。

如果已選取此型樣參數，則必須設定錯誤佇列管理程式及錯誤佇列的值。

錯誤佇列管理程式 此型樣參數定義用於錯誤記載的佇列管理程式。

只有在選取需要錯誤訊息時，才需要此型樣參數。

錯誤佇列 此型樣參數定義用於錯誤訊息的佇列。

只有在選取需要錯誤訊息時，才需要此型樣參數。

所有型樣參數會與其預設值一起列出。 如果未指出任何預設值，則預設值是空字串。
