使用 Microsoft Dynamics CRM Account Entity 輸出：動態傳輸輸入型樣的限制

當您使用此型樣時，將適用一些限制。

僅當您符合所有下列準則時，才可以使用 Microsoft Dynamics CRM Account Entity 輸出：動態傳輸輸入型樣：

  • 型樣只能在 Microsoft Windows 環境中的分配管理系統上部署及執行，因為 .NETCompute 節點取決於 .NET Framework 與 Microsoft Dynamics CRM Systems 的連線功能。
  • 訊息流程實例與 BAR 檔包含明碼 Microsoft Dynamics CRM 連線認證。如果這不適合安全需求，則您必須修改訊息流程，以便 .NETCompute 節點能夠依安全機制取得這些詳細資料。如需相關資訊，請參閱套用 Microsoft Dynamics CRM Account Entity 輸出：動態傳輸輸入型樣之前要完成的作業。
  • 訊息流程實例只能設定及取得 Microsoft Account Entity 上的標準屬性。您可以修改並延伸自訂屬性的型樣建立的流程實例。如需相關資訊，請參閱套用 Microsoft Dynamics CRM Account Entity 輸出：動態傳輸輸入型樣之前要完成的作業。
